# README

## Sorbet Typing

### 1. Install Sorbet
Add this gems to Gemfile development block:
```
gem 'sorbet-runtime'

# In development block:
gem 'sorbet'
gem 'tapioca'
```

### 2. Run tapioca
To generate initial files run:
```
bundle exec tapioca init
```
This will generate `sorbet/` subdirectory with the rails .rbi files
